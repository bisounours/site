Nom : @Tension (2009)
Langages utilisés : PHP, HTML, Javascript(JQuery), CSS, MySQL
Description : Cette application permet de saisir la tension systolique et diastolique ainsi que la pulsion afin d'effectuer un suivi journalier.
En plus de ce suivi, il est possible d'effectuer des statistiques mensuelles et annuelles ainsi que des exports en PDF.

Ce projet m'a été inspiré par des personnes de mon entourage ayant des problèmes de tension.
Le principal intêret de cette application est de garder une trace des mesures prises à l'aide d'un tensiomètre et de faciliter le suivi éventuel d'un médecin.

Bilan : Ce projet m'a permis d'utiliser une librairie javascript pour les graphiques (JQPlot) ainsi qu'une libraire de création de PDF en PHP (FPFD).
Cette application web pourrait largement être améliorée.
Nom : LittleCupCake (2012)

Description : Ce projet est mon premier site web "graphique", il s'agit d'un site de recette de cuisine assez basique composé uniquement de page de recette avec la possibilité de noter la recette et d'y laisser un commentaire.
