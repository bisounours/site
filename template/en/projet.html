Name : @Tension (2009)
Languages used : PHP, HTML, Javascript(JQuery), CSS, MySQL
Description : This application allows you to enter the systolic, diastolic and blood pressure as well as the drive to perform daily tracking.
In addition to this monitoring, it is possible to make monthly and annual statistics and export to PDF.

This project was inspired by people around me having health problems. The main interest of this application is to keep track of the steps taken using a sphygmomanometer and facilitate possible follow a doctor.

Result : This project allowed me to use a javascript library for graphics (JQPlot) and a bookseller PDF creation with PHP (FPFD).
This web application could be largely improved.
